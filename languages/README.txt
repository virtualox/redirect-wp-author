This directory is for translation files (.po and .mo files).

The plugin includes translations for the following languages:
- Dutch (nl_NL)
- German (de_DE)
- French (fr_FR)
- Russian (ru_RU)
- Ukrainian (uk)
- Spanish (es_ES)
- Japanese (ja)
- Portuguese (pt_PT)
- Persian/Farsi (fa_IR)
- Turkish (tr_TR)
- Chinese (Simplified) (zh_CN)

To create additional translations for this plugin:
1. Extract the translatable strings using WordPress i18n tools
2. Create .po files for each language
3. Generate .mo files from the .po files
4. Place them in this directory following the naming convention:
   redirect-wp-author-{locale}.mo and redirect-wp-author-{locale}.po

Example: redirect-wp-author-nl_NL.mo for Dutch translations.
